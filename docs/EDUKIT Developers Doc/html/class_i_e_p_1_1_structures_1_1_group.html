Предупреждения
Конкретно в этом классе переопределён метод __call(). Это было сделано для того, чтобы убрать "паравозик" при вызове методов из объекта
Например, чтобы раньше получить код специальности, нужно было писать: $group->getSpec()->getCode()
После переопределения метода _call() можно писать: $group->getCode()
В обоих случаях мы получаем код специальности
